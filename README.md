[![License](https://img.shields.io/github/license/cligpt/shres.svg)](https://github.com/cligpt/shres/blob/main/LICENSE)



## Introduction

*shres* is the resources of [shai](https://github.com/cligpt/shai).



## Usage

```bash
# Init config
./init.sh

# Make resource
./make.sh

# Build with .syso
./build.sh
```
